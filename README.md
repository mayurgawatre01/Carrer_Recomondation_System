# Career Recommendation System 🚀

An AI-powered career recommendation system that helps users explore suitable career paths based on skills, interests, and input preferences.

## Features
- Career recommendation
- User input analysis
- Skill-based suggestions
- Simple UI
- Fast prediction

## Tech Stack
- Python
- HTML
- CSS
- Flask
- Machine Learning

## Project Structure

bash
project/
│── static/
│── templates/
│── app.py
│── model/
│── requirements.txt


## How to Run

1. Clone repository

bash
git clone <repo-url>


2. Install requirements

bash
pip install -r requirements.txt

3. Run

bash
python app.py
